Let's Set Up The Environment

Step 1: Download The Setup

This download contains the GCC Compiler to run your code smoothly.

Download Link 1 Download Link 2

Step 2: Install The Setup

1. Once You Download. Install The Software.

2. Click on Install, Then Just click on Continue...... Software Will Installed.

3. Click on "Basic Setup", Then Right Click on "mingw32-gcc-objc"

Click on "Mark fot Installaltion"

4. Go To "Installaltion" and Click on "Apply Changes", Then Click on "Apply".

Setup Is Sccessfully Installed. Now You Can Click on "Close"

Step 3: Setting Up the PATH

1. Copy This Path "C:\MinGW\bin" And Paste In "Environment Variable" --> "Path".

2. Select "Path" in "System Variable" Double Click on "Path" Or Click on "Edit".

3. Click on "New" and Paste the Path = "C:\MinGW\bin"

4. Just CLick on "OK" And Then Close


You Have Sccessfully Installed The Software
